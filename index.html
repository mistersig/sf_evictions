Evictions filed in San Francisco, 2007-2017

For this project I decided to use D3 and have a map work in parallel with a bar chart. This was my first time using D3js and it was painful at times, but I got a working application. The data is eviction types filed in San Francisco between, 2007-20117. Disclaimer, not all eviction notices resulted in eviction. The city of San Francisco has 15 types classified. See the data source for more information.
